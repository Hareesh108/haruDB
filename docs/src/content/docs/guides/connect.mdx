title: Connect
description: Connect to HaruDB using Telnet or the CLI client.
---

## Telnet

```bash
telnet localhost 54321
```

## Haru CLI

```bash
# Connect to default port
haru-cli

# Connect to custom port
haru-cli --port 54322

# Connect to remote host
haru-cli --host 192.168.1.100 --port 54321
```
